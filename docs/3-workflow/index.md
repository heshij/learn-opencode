# 第三阶段：高效工作流

> 💡 **目标**：掌握 Plan/Build 模式、Agent 协作、项目初始化

## 🎯 阶段目标

- 理解 Plan 模式和 Build 模式的区别
- 学会调用不同 Agent 完成任务
- 用 `/init` 让 AI 读懂你的项目

## ⏳ 预计耗时

45 分钟 ~ 1 小时

## 🚀 开始之前

- 完成了第一、第二阶段
- 有一个真实项目（代码或写作项目都行）

## 📚 本阶段课程

| 课程 | 内容 | 时长 |
|-----|------|------|
| [3.1 Plan vs Build](./01-plan-build) | 规划模式与执行模式 | 15 分钟 |
| [3.2 认识 Agent](./02-agents) | 你的专属 AI 团队 | 15 分钟 |
| [3.3 项目初始化](./03-init) | 让 AI 读懂你的项目 | 15 分钟 |

## 🎒 你将收获

完成本阶段后，你将：

1. 知道什么时候用 Plan 模式，什么时候用 Build 模式
2. 能调用不同 Agent 完成专业任务
3. 能用 `/init` 生成项目规则文件
