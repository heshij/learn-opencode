# 第一阶段：快速起步

> 💡 **目标**：30 分钟内完成安装并成功对话

## 🎯 阶段目标

- 成功安装 OpenCode 且不报错
- 搞定网络连接（国内用户重点）
- 完成第一次 "Hello World" 对话

## ⏳ 预计耗时

25 ~ 45 分钟

## 🚀 开始之前

- 一台能上网的电脑（Windows / macOS / Linux）
- 终端应用（推荐 WezTerm / iTerm2 / Windows Terminal）
- 一杯水 ☕

完成本阶段后，你将：

1. 在终端里运行 `opencode` 命令
2. 配置好网络（或选择国内可直连的国产模型）
3. 和 AI 进行第一次对话
