这次更新让 **ChatGPT Plus/Pro 用户可以直接登录使用 GPT-5.2-Codex 模型**，不需要单独买 API，同时修复了长时间使用变卡的问题。

## 新功能：GPT-5.2-Codex 支持

### 这次更新了什么？

OpenAI 最新的代码模型 **GPT-5.2-Codex** 现在可以在 OpenCode 中使用了。

**最重要的是**：如果你已经是 ChatGPT Plus（$20/月）或 Pro（$200/月）用户，可以**直接用订阅登录**，不需要单独购买 API！

### GPT-5.2-Codex 是什么？

GPT-5.2-Codex 是 OpenAI 在 2025年12月18日发布的**专业代码模型**，基于 GPT-5.2 进一步优化。它特别擅长：

- 长时间代码任务（支持上下文压缩）
- 大规模重构和代码迁移
- 理解设计稿和截图
- 漏洞发现和代码审计

### 什么时候用 Codex？

| 场景 | 推荐模型 |
|------|----------|
| 写复杂功能、重构代码 | ✅ GPT-5.2-Codex |
| 日常问答、写文档 | GPT-5.2 Instant |

**简单说**：写代码用 Codex，其他用 GPT-5.2 Instant。

### 如何使用？

👉 **完整教程**：[连接 OpenAI（GPT-5.2 / Codex）](/1-start/04h-openai)

**快速体验**（ChatGPT Plus/Pro 用户）：

1. 输入 `/connect`
2. 选择 **OpenAI**
3. 选择 **ChatGPT Plus/Pro** 登录方式
4. 浏览器授权完成
5. 输入 `/models`，选择 `gpt-5.2-codex`
6. 开始写代码！

## 问题修复：长时间使用不再卡顿

### 之前的问题

有用户反馈：OpenCode 运行几个小时后，界面会变得越来越慢，甚至卡死。

### 现在的情况

这个问题已经修复。你可以放心地让 OpenCode 一直开着，不用担心变卡。
